### Running locally

Clone the project

```bash
  git clone https://github.com/Pedro-Toselli/easy-darkmode-example.git
```

Enter the project directory

```bash
  cd easy-darkmode-example
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
